# Documentação Comercial - Base Exata

Esta pasta reúne materiais práticos para usar em reuniões, diagnósticos, propostas e apresentações comerciais da Base Exata.

## Como usar esta pasta

1. Use `01-diagnostico-gratuito.md` quando quiser oferecer uma primeira conversa estruturada, sem pressão comercial.
2. Use `02-roteiro-reuniao-com-cliente.md` para conduzir reuniões comerciais com foco em entendimento do negócio.
3. Use `03-checklist-levantamento-de-dados.md` para mapear quais dados o cliente possui e como eles são usados hoje.
4. Use `04-modelo-proposta-comercial.md` como base editável para enviar propostas.
5. Use `05-pacotes-de-servico.md` para apresentar formatos de entrega mais comuns.
6. Use `06-perguntas-frequentes.md` para responder dúvidas recorrentes de forma simples.
7. Use `07-objeções-e-respostas.md` para tratar resistências sem pressionar o cliente.
8. Use `08-roteiro-apresentacao-demos.md` para conduzir demos do portfólio conforme a dor do cliente.

## Observações importantes

- As demos do repositório usam dados simulados e servem para demonstrar cenários de negócio.
- Os materiais desta pasta são comerciais e consultivos. Eles não substituem contrato, termo jurídico ou definição técnica detalhada.
- Ajuste cada documento ao contexto do cliente antes de usar em reunião ou envio.
